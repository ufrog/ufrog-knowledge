# 查看端口占用

在开发时本地启动服务，往往会发生端口已被占用，无法启动服务的情况，这个时候我们就需要找到具体哪个应用占用了相关端口，并进行处理。

首先我们打开命令行，一般我们需要打开`CMD`，或者可以使用`PowerShell`，再有也可以在IDEA的`Terminal`的窗口中运行。

输入命令`netstat -ano|findstr "xxxx"`（其中xxx替换成被占用的端口号）可以得到如下响应：

![alt 端口占用响应](/images/20200616/20200616100648.png)

圈起来的就是系统进程`PID`了，这个时候可以通过命令查看相关`PID`对应的进程`tasklist|findstr "xxxx"`（xxxx替换成查询出来的PID）

![alt PID查询进程](/images/20200616/20200616103705.png)

可以看到这个PID对应的是一个JAVA进程。之后我们只需要`Ctrl + Alt + Delete`进入任务管理器，找到相关PID的进程，是不是要杀掉就看你自己的了。
