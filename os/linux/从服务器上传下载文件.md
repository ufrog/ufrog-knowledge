# 从服务器上传/下载文件
经常我们需要从「Linux」服务器上下载文件到我们「Windows」开发机上，也经常需要从「Windows」开发机上传文件到「Linux」服务器上，这里就总结几个方法以供参考。
## 1. SCP
「SCP」可以说是最简单的方法，因为`scp`命令是「Windows」自带的，我们只需要打开「CMD」或者「PowerShell」就可以直接使用，如下命令均为「Windows」下执行。
### 从服务器上下载文件
```
# 命令格式
scp ${account}@${ip}:${path_on_linux} ${windows_path}
# 举个栗子
scp root:127.0.0.1:/usr/local/file.txt C:\Users\Admin\Downloads
```
### 上传到服务器
```
# 命令格式
scp ${windows_path} ${account}@${ip}:${path_on_linux}
# 举个栗子
scp C:\Users\Admin\Downloads\file.txt root:127.0.0.1:~/file
```
