# 修改密码
有的时候为了安全考虑，我们需要定期修改账号密码，其实修改密码很简单。如果修改当前用户密码，直接执行命令，会要求你输入原密码和新密码
```shell
passwd
Changing password for user {USER}.
Changing password for {USER}.
(current) UNIX password: 
New password: 
Retype new password: 
passwd: all authentication tokens updated successfully.
```
如果是`root`用户，需要修改其他用户的密码，可以用命令
```shell
passwd <user_name>
```
其他的操作是一样的
